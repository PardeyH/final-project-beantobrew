Bean to Brew

Bean to Brew
is a game that teaches you about the coffee making process.

Gameplay: Move the cursor over the pictures to learn about the coffee making process from harvesting the coffee beans to pouring freshly ground coffee into your cup. Once you have read all the information you can answer the question and if correctly answered move on to the next level.

Scoring: For each correct answer you get points.

Winning: You win the game by answering all the questions correctly.


Starting the game: Please login or register to start the game. Happy brewing!
